Merge

Git merge combines two branches into one. It does this by creating a new commit which combines the work from the 2 commits specified. In image 1, you can see 2 branches, bugfix and master. Notice that master is the current checked out commit. By typing git merge bugfix into the command line, we get a new commit, C4, and the master branch now encompasses bugfix as seen in image 2.
