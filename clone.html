Clone

Git clone clones your local repository into a remote one. In image 1, we see that there is a local master repository. After typing in “git clone” into your terminal, you will see that a remote clone has been created and there are now 2 repositories (image 2). Also, git clone is commonly used to download a local copy of a remote repository to your machine. For example, if there is a repository on GitHub that you want to work on locally, you would use git clone to “clone” it to your machine.
